## 📋 项目总览

| 维度 | 内容 |
|------|------|
| **目标** | 验证 FB 流量获取 Agent 可行性 |
| **周期** | 4 周（Week 1-4） |
| **预算** | $18,000-20,000 |
| **成功标准** | 获客成本<$3/申请，转化率>2%，账号存活>10 天 |
| **失败条件** | 连续 4 周成本>$5、封号率>80%、收到律师函、投诉率>5% |

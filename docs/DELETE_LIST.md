## 📝 删除规范

### 什么情况下应该删除文档？

| 情况 | 建议 |
|------|------|
| 文档内容过时 | ✅ 删除或移到 archive/ |
| 文档定位错误 | ✅ 删除或重构 |
| 文档重复 | ✅ 删除冗余版本 |
| 临时草稿 | ✅ 完成后删除 |
| 重要历史文档 | ⚠️ 移到 archive/，不要删除 |

### 删除前检查清单

- [ ] 确认文档没有业务价值
- [ ] 确认没有其他文档引用此文档
- [ ] 确认不是重要历史记录（如果是，移到 archive/）
- [ ] 已在 DELETE_LIST.md 中记录删除原因

## 一、页面总览

### 1.1 页面列表

| 页面 | 路径 | 优先级 | 描述 |
|:-----|:-----|:------:|:-----|
| **Dashboard** | `/` | P0 | 数据看板首页 |
| **Agent 列表** | `/agents` | P0 | Agent 管理与新建 |
| **Agent 详情** | `/agents/:id` | P1 | Agent 详情与配置 |
| **工作流列表** | `/workflows` | P0 | 工作流管理 |
| **工作流编辑器** | `/workflows/:id/edit` | P0 | YAML 可视化编辑 |
| **执行记录** | `/executions` | P0 | 执行历史查询 |
| **执行详情** | `/executions/:id` | P1 | 执行日志与进度 |
| **系统设置** | `/settings` | P2 | 系统配置管理 |

### 1.2 信息架构

```
Hunter Agent 管理系统
├── Dashboard (数据看板)
├── Agent 管理
│   ├── Agent 列表
│   └── Agent 详情
├── 工作流管理
│   ├── 工作流列表
│   ├── 工作流编辑器
│   └── 执行记录
└── 系统设置
    ├── 系统配置
    └── 权限管理
```

## 三、交互设计

### 3.1 状态标识

| 状态 | 图标 | 颜色 | 说明 |
|:-----|:----:|:-----|:-----|
| **Online** | 🟢 | 绿色 | Agent 在线，可执行任务 |
| **Offline** | 🔴 | 红色 | Agent 离线，心跳超时 |
| **Busy** | 🟡 | 黄色 | Agent 忙碌，正在执行任务 |
| **Draft** | 📝 | 灰色 | 工作流草稿，未发布 |
| **Published** | ✅ | 绿色 | 工作流已发布 |
| **Archived** | ⚠️ | 橙色 | 工作流已归档 |
| **Pending** | ⏸ | 灰色 | 执行等待中 |
| **Running** | ⏳ | 蓝色 | 执行进行中 |
| **Completed** | ✅ | 绿色 | 执行完成 |
| **Failed** | ❌ | 红色 | 执行失败 |
| **Cancelled** | ⏹ | 灰色 | 执行已取消 |

### 3.2 操作反馈

| 操作 | 成功反馈 | 失败反馈 |
|:-----|:---------|:---------|
| 新建 Agent | ✅ Agent 新建成功 | ❌ Agent 新建失败：{原因} |
| 删除 Agent | ✅ Agent 已删除 | ❌ 删除失败：有关联工作流 |
| 创建工作流 | ✅ 工作流创建成功 | ❌ 创建失败：YAML 格式错误 |
| 发布工作流 | ✅ 工作流已发布 | ❌ 发布失败：存在无效步骤 |
| 执行工作流 | ✅ 执行已启动 (exec_XXX) | ❌ 执行失败：Agent 离线 |
| 重试执行 | ✅ 重试已启动 (exec_XXX) | ❌ 重试失败：状态不允许 |
| 取消执行 | ✅ 执行已取消 | ❌ 取消失败：已完成 |
| 保存配置 | ✅ 配置已保存 | ❌ 保存失败：权限不足 |
